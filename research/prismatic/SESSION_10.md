# Session 10: Cohomology of thickened dual numbers F_2[s]/(s^d)

**Дата**: 2026-04-22
**Цель Session 9 переданная**: verify conjecture H¹(F_2[s]/(s^d)) = Z/d.

## Главный результат — THEOREM

**Theorem (Session 10)**: Для R_d = F_2[s]/(s^d) над Z_2:

$$H^0_{dR}(R_d) = \mathbb{Z}_2$$

$$H^1_{dR}(R_d) = \bigoplus_{k=1,3,5,\ldots}^{d-1} \mathbb{Z}/2^{v_2(k+1)}$$

где v_2(m) = 2-adic valuation of m.

Для d = 2^j: **|H¹_dR(R_d)| = 2^{d-1}**.

## Доказательство

**Setup**: R_d = Z_2[s]/(s^d). Z_2-basis {1, s, ..., s^{d-1}}.

**Kähler differentials**:
- Ω¹ generated by ds с relations from d(s^d) = 0
- d(s^d) = d·s^{d-1}·ds = 0 в Ω¹
- Ω¹ = R_d·ds / (d·s^{d-1}·ds)
- As Z_2-module: Ω¹ = ⊕_{k=0}^{d-2} Z_2·s^k ds ⊕ (Z/d)·s^{d-1} ds

**Differential** d: R_d → Ω¹:
- d(s^k) = k·s^{k-1}·ds
- Coefficient на position k в Im(d) = (k+1)·Z_2 ⊆ Z_2

**H¹ по позициям**:
- Position k (k ≤ d-2): Z_2 / (k+1)Z_2 = Z/2^{v_2(k+1)}
  - При v_2(k+1) = 0 (т.е. k+1 odd, k even): quotient = 0
  - При v_2(k+1) > 0 (k odd): non-trivial
- Position d-1: Z/2^{v_2(d)} (from truncation, no image from R_d)

**Total**: H¹ = ⊕_{k odd, 1 ≤ k ≤ d-1} Z/2^{v_2(k+1)} ⊕ (Z/2^{v_2(d)} if k = d-1 even)

Для d = 2^j, позиция d-1 odd → uже включена.

**Order для d = 2^j**:
- Odd k ∈ [1, d-1]: d/2 значений
- Sum of exponents = Σ_{k odd} v_2(k+1) = Σ_{m=1}^{d/2} v_2(2m) = d/2 + Σ_{m=1}^{d/2} v_2(m)
- Σ v_2(m) для m=1..d/2 = 2^{j-1} = v_2((d/2)!) = 2^{j-1} - 1 (Legendre для p=2)
- Total exponent: 2^{j-1} + 2^{j-1} - 1 = 2^j - 1 = d - 1

QED.

## Connection to SHA-256

**SHA-256 uses 32-bit rotations** ⇒ rotation ring = F_2[s]/(s^16) (from Session 9 identification).

**Our theorem для d=16**:
$$H^1(F_2[s]/(s^{16})) = \mathbb{Z}/2 \oplus \mathbb{Z}/4 \oplus \mathbb{Z}/2 \oplus \mathbb{Z}/8 \oplus \mathbb{Z}/2 \oplus \mathbb{Z}/4 \oplus \mathbb{Z}/2 \oplus \mathbb{Z}/16$$

Order: 2¹⁵ = 32768. 8 cyclic factors.

**Full SHA-256 state** = 8 registers × 32-bit.
H¹ для "rotation ring of full state" = (Z/2¹⁵)^8 (roughly).
Total order: 2¹⁵×⁸ = 2¹²⁰.

**Сравнение с SHA-256 security**:
- Birthday bound: 2¹²⁸
- Наш H¹: 2¹²⁰
- **Разница 2⁸ бит** — не достаточно чтобы пробить

## Интерпретация — что мы получили

**Concrete finite invariant** SHA rotation structure: (8 × abelian group of order 2¹⁵) = 2¹²⁰ classes.

Но это только **rotation part**. Полный SHA включает:
- Rotations (Σ₀, Σ₁, σ₀, σ₁) — characterized by this cohomology
- XOR/AND (Ch, Maj) — formula из Sessions 2-3
- ADD — respects δ (Session 2)

**Integrated cohomology** SHA round function должна combine эти три.

## Открытые вопросы для Session 11+

### Q1: Multi-variable extension
Sessions 9-10 посчитали F_2[s]/(s^d) — single variable. SHA имеет 8 registers. Нужно: **F_2[s_1, ..., s_8]/(s_i^{16})** — tensor product of 8 copies.

By Künneth formula: H¹(⊗ R_i) = ⊕ H¹(R_i). Так:
H¹((F_2[s]/(s^16))^⊗8) = ⊕_8 H¹(F_2[s]/(s^16)) = 8 × (Z/2 ⊕ ... ⊕ Z/16)

Order: 2¹²⁰. Matches earlier estimate.

### Q2: AND/XOR/ADD integration
F_2[s]/(s^16) captures rotation. Как ADD (ring operation), XOR (not ring), AND (primitive) взаимодействуют?

Мы имеем:
- XOR formula (Session 2) — δ-discrepancy
- AND ANF degree pattern (Sessions 3-4) — structural
- ADD compatibility (Session 6)

**Integration**: compute **relative** cohomology того ring, где все операции живут, но не только rotation ring.

### Q3: Full SHA round function as morphism
Ultimate goal: round function f: state → state как **morphism** в подходящей category. Induces map on cohomology. Structural invariant of SHA = что сохраняется этим map.
