# Session 9: Prismatic cohomology of dual numbers F_2[ε]/(ε²)

**Дата**: 2026-04-22
**Цель Session 8 переданная**: compute prismatic cohomology of F_2[ε]/(ε²).

## Главные результаты

### 1. Z_2[ε]/(ε²) IS a δ-ring (verified)

Frobenius lift: φ(ε) = 0, φ(a + bε) = a (drops ε).

Verified D1, D2, D3 axioms empirically on test pairs:
- D1: δ(0) = δ(1) = 0 ✓
- D2: δ(x+y) = δ(x) + δ(y) - xy ✓ (4 test pairs)
- D3: δ(xy) = x²δ(y) + y²δ(x) + 2δ(x)δ(y) ✓ (4 test pairs)

**Conclusion**: дуальные числа Z_2[ε]/(ε²) — valid δ-ring lift over Z_2 of F_2[ε]/(ε²).

### 2. Computed Kähler differentials Ω¹

**Setup**: R = Z_2[ε]/(ε²) over Z_2, generated by ε with relation ε² = 0.

**Kähler differentials**:
- Ω¹ generated by dε
- Relation: d(ε²) = 2ε·dε = 0
- Ω¹ = R·dε / (2ε·dε)
- = Z_2·dε ⊕ (Z_2/2Z_2)·ε dε
- ≅ **Z_2 ⊕ F_2** as Z_2-module

### 3. Computed de Rham cohomology

Differential d: R → Ω¹, d(a + bε) = b·dε.
- Im(d) = Z_2·dε
- Ker(d) = Z_2 (constants)

**de Rham cohomology**:
$$H^0_{dR} = \ker(d) = \mathbb{Z}_2$$
$$H^1_{dR} = \Omega^1 / \text{Im}(d) = F_2 \cdot \varepsilon\, d\varepsilon \cong F_2$$
$$H^i_{dR} = 0 \text{ for } i \geq 2$$

**Это первая нетривиальная когомологическая группа** в нашей программе! H¹ = F_2 ≠ 0.

### 4. Prismatic cohomology relationship

Для SMOOTH R/k: prismatic ≅ de Rham (modulo Hodge-Tate filtration).

Дуальные числа **NOT smooth** — singular point at ε=0. Для non-smooth case prismatic может иметь:
- Дополнительные torsion classes
- Derived enhancements
- Conjugate filtration corrections

**Best concrete approximation на session level**: prismatic ≈ de Rham + maybe extra torsion in higher degrees.

**Conjectured H^* of F_2[ε]/(ε²) prismatic**:
- H⁰_prism = Z_2 (matches de Rham)
- H¹_prism = F_2 (matches de Rham)
- H^i_prism for i ≥ 2: possibly extra torsion (uncertain без full Bhatt-Scholze machinery)

## Связь с SHA — корректировка

После пересчёта (correction from session 8):

**Rotation rings для n-bit registers** (n = 2^k):
- n = 4 (k=2): F_2[ζ_4] = F_2[s]/(s²) = **dual numbers** (наша computation!)
- n = 8 (k=3): F_2[s]/(s⁴)
- n = 16 (k=4): F_2[s]/(s⁸)
- n = 32 (k=5): F_2[s]/(s^16) ← **SHA-256 32-bit registers**

Pattern: **bit width n = 2^k ⇒ rotation ring = F_2[s]/(s^{n/2})**.

(Через substitution s = ζ_n + 1, и факт что Φ_{2^k}(x) = x^{2^{k-1}} + 1 = (x+1)^{2^{k-1}} mod 2.)

So:
- Наша Session 9 computation H¹ = F_2 для F_2[s]/(s²) — соответствует **4-bit rotation case**
- Для full SHA-256 нужно F_2[s]/(s^16)

## Conjectural pattern для F_2[s]/(s^d)

Для F_2[s]/(s^d) аналогичная computation:
- Ω¹ = R·ds / (d·s^{d-1}·ds) ... (relation from d(s^d) = d·s^{d-1}·ds = 0)
- For d = 2: 2s·ds = 0, gives F_2 torsion
- For d = 4: 4s³·ds = 0; in Z_2 the coefficient 4 = 2², gives Z/4 torsion
- For d = 8: coefficient 8 = 2³, gives Z/8 torsion
- For d = 16: coefficient 16 = 2⁴, gives Z/16 torsion
- For d = 2^j: coefficient 2^j, gives Z/2^j torsion

**Conjecture**: H¹ of F_2[s]/(s^{2^j}) = Z/2^j (Z_2-module).

Для SHA-256 (d=16, j=4): conjectured H¹ = **Z/16 = Z/2⁴**.

This is **non-trivial 4-bit torsion class** for each rotation register. **Полностью SHA-state** имеет 8 registers × 32-bit = 8 instances of this.

## Что это означает для атаки SHA?

Если conjecture верна, prismatic cohomology of "rotation part of SHA" имеет:
- 8 копий Z/16 в H¹ (one per register)
- Total: (Z/16)⁸ = group of order 16⁸ = 2³² classes

**Это потенциально detectable invariant**, но:
- Не очевидно как использовать для коллизий
- 32-bit invariant — не достаточно для пробивания birthday 2¹²⁸
- Требует дальнейшего исследования
