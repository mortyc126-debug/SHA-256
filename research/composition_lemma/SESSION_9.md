## 1. Computation

Two independent tests applied к D^4 (dim 264):

### Test A: 30 random elements (Session 9 v1)

Random x ∈ D^4 (uniform over F_2^264). Compute smallest ideal containing x.
**Result**: All 30 trials → ideal dim = 264 (full D^4).

**Initial interpretation** (Session 9 v1): "Strong evidence D^4 simple."

**Corrected interpretation** (Session 9b): random elements are GENERIC — each random x has component in many minimal ideals → ideal closure absorbs all of them → full D^4. Random test НЕ может detect proper ideals if they are "small directions".

### Test B: All 264 basis elements deterministically (Session 9b)

For each basis element x_i (i = 0..263), compute ideal generated by x_i.

**Result** (47 of 264 basis elements gave proper ideals):

| Dim of ideal | Count of basis elements |
|---|---|
| 45 | 17 |
| 55 | 22 |
| 65 | 5 |
| 75 | 3 |
| 264 (full) | ~217 |

Specific positions: i ∈ {0, 1, 3, 5, 8, 12, 17, 24, 25, 26, 29, 30, 33-37, 41, 44, 54, 63, 64, 68, 70, 71, 74, 76, 90, 91, 95, 97, 104, 112, 120, 121, 135, 138, 155, 157, 160, 162, 170, 190, 197, 216, 219, 256}.

**Pattern**: small-index basis elements (i ≤ 90) tend to give smaller ideals (45, 55). Larger-index (i ≥ 100) tend to give larger (65, 75).

Computational cost: ~900s for 264 basis elements (slower than estimated, but feasible).

## 2. Interpretation

### 2.1 D^4 is NOT simple

Direct conclusion: existence of proper nontrivial ideal refutes simplicity. **Conjecture 21.1 in strong form (D^4 itself simple) REFUTED.**

### 2.2 Conjecture 21.1 in weak form still open

D^4 may still be semisimple = direct sum of simple Lie algebras. The proper ideals found are candidates для simple summands.

Observed dims {45, 55, 65, 75} — все multiples of 5. Notable arithmetic property:
- 45 = 9 · 5
- 55 = 11 · 5
- 65 = 13 · 5
- 75 = 15 · 5

Sum of all distinct? 45 + 55 + 65 + 75 = 240 ≠ 264. So either:
(a) These ideals overlap (not direct summands),
(b) There are MORE distinct ideals не yet found,
(c) Some are not minimal.

### 2.3 Structural hint

Pattern "small i → small ideal" suggests:
- D^4 basis was constructed via iterated brackets from 4 generators (Σ_0 - I, Σ_1 - I, σ_0 - I, σ_1 - I).
- Earlier basis elements (small i) correspond к short bracket chains.
- These generate "shallow" sub-structure of D^4.
- Possibly: short brackets generate ideal containing only "low-order" interactions; deep brackets bring in full algebra.

This is consistent с D^4 having **filtration** или **gradation** structure, where ideals correspond к specific "depths" of bracket complexity.

## 4. Cryptanalytic implications (preliminary)

**If D^4 = ⊕ L_i** with L_i acting on V_i ⊆ F_2^32:

Each V_i is **invariant subspace** of SHA's linear layer (Σ + σ + identity). This is a **structural decomposition** F_2^32 = ⊕ V_i (or some reducible covering).

**Possible exploit angle**:
- Bits within same V_i interact only within their L_i orbit.
- Bits across different V_i have NO linear-layer interaction.
- IF nonlinear ops (Ch, Maj, ADD, σ message schedule) **respect** decomposition (which they don't for full SHA, but maybe partially) → SHA effectively reduces to product of smaller hashes on V_i's.

**Realistic assessment**: nonlinear ops в SHA по design mix all bits через ADD carry. Decomposition almost certainly broken by these. But **partial respect** would give weak-input class где SHA acts as smaller hash on subspace.

This is similar к Φ-manifold finding (6D substructure) — decomposition exists, but не controllable from input → ⊘SCOPED. The SAME analysis must be applied к L_i decomposition.

**Key question for Session 11**: do decomposition subspaces V_i admit input control? If MI(W; V_i specific bits) > 0 with significant signal, exploit possible. Если MI ≈ 0 (как для Φ) → ⊘SCOPED also.
