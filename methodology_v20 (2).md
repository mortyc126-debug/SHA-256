## 2. АРХИТЕКТУРА SHA-256 — КЛЮЧЕВЫЕ СВОЙСТВА

### 2.1 Раундовая функция

SHA-256: 64 раунда, 8 регистров (a,b,c,d,e,f,g,h) по 32 бита, 16 входных слов W[0..15].

```
T1 = h + Σ1(e) + Ch(e,f,g) + K[r] + W[r]
T2 = Σ0(a) + Maj(a,b,c)
(a,b,c,d,e,f,g,h) ← (T1+T2, a, b, c, d+T1, e, f, g)
```

- Σ1(e) = ROTR⁶(e) ⊕ ROTR¹¹(e) ⊕ ROTR²⁵(e)
- Σ0(a) = ROTR²(a) ⊕ ROTR¹³(a) ⊕ ROTR²²(a)
- Ch(e,f,g) = (e & f) ⊕ (~e & g) — выбор, билинейна
- Maj(a,b,c) = (a & b) ⊕ (a & c) ⊕ (b & c) — большинство, билинейна

### 2.2 Message Schedule

```
W[t] = σ1(W[t-2]) + W[t-7] + σ0(W[t-15]) + W[t-16]   (t = 16..63)
```

- σ0(x) = ROTR⁷(x) ⊕ ROTR¹⁸(x) ⊕ SHR³(x)
- σ1(x) = ROTR¹⁷(x) ⊕ ROTR¹⁹(x) ⊕ SHR¹⁰(x)
- σ0, σ1 **линейны над GF(2)**, нелинейны над Z/2^32 (из-за +)
- Матрица XOR-расписания L: GF(2)^512 → GF(2)^1536 имеет **ранг 512/512** (полный)

### 2.3 Асимметрия e/a-ветвей

```
e_new = d + T1    — ДВА независимых источника (d из a-сдвига, T1 из раунда)
a_new = T1 + T2   — T2 = Σ0(a) + Maj(a,b,c) зависит от a → САМОРЕФЕРЕНЦИЯ
```

- Wang chain работает на e-ветви (есть внешний источник через d)
- Wang chain НЕ работает на a-ветви (T2=0 при δa=0 → тривиально)
- δa и δe **ортогональны** (T_DA17_DE17_ORTHOGONAL, подтверждено 20M пар)
- Инвариант φ = δa - δe **не зависит от schedule** (T1 сокращается)

### 2.4 Три фазы распространения дифференциала

| Фаза | Раунды | Характеристика |
|------|--------|----------------|
| 1: Разомкнутая | r=1..3 | Δd=0, Δh=0; De зависит только от Σ1, Ch, W |
| 2: Активация эха | r=4..6 | Δd₄=Δa₁, Δh₄=Δe₁; резкий рост HW(De) |
| 3: Насыщенная | r≥7 | Δd и Δh оба ~8-16 бит; HW стабилизируется |

### 2.5 TLC-декомпозиция (Three-Layer Collision Theory)

Дифференциал SHA-256 раскладывается **точно**:

```
ΔH = L(ΔM) ⊕ Q(M, ΔM) ⊕ C(M, ΔM)
```

| Слой | Алгебра | Вклад | Природа |
|------|---------|-------|---------|
| **L** — линейный | GF(2) | 25.8% | Σ0, Σ1, σ0, σ1, XOR, schedule |
| **Q** — квадратичный | GF(2), deg=2 | 25.3% | Ch, Maj (билинейные) |
| **C** — carry | Z_{2^32} | 48.9% | Carry chains в mod-add |

39 верифицированных фактов. Три независимых барьера: Schedule (rank 512/512), Carry (rank 256/256), Quadratic (SAT timeout 32R+).

## 6. CARRY THEORY — НОВЫЙ РАЗДЕЛ МАТЕМАТИКИ

### 6.1 Аксиоматика

Carry system CS(B) = (G, B, φ, c):
- G = digit group, B = base, φ = digit addition, c = carry function
- C1: φ(a,b) + B·c(a,b,0) = a+b in Z
- C2: c_i = c(digit_i(a), digit_i(b), c_{i-1}), c_{-1} = 0

### 6.2 Carry Algebra {G, K, P, ?}

Полугруппа с таблицей композиции:

|   | G | K | P | ? |
|---|---|---|---|---|
| G | G | G | G | G |
| K | K | K | K | K |
| P | G | K | P | ? |
| ? | G | K | ? | ? |

### 6.3 Carry Operator

```
C(a,b) = (a+b - a⊕b)/2

C(a,0) = 0, C(a,a) = a, C(a,~a) = 0, C(a,b) = C(b,a)
|Z(C)| = 3^n (carry-free pairs, P = (3/4)^n)
C is idempotent: C(C(a,b), C(a,b)) = C(a,b)
```

### 6.4 Carry Calculus

```
∂C/∂a_k: lower-triangular matrix
  [k] = b_k ⊕ c_{k-1} (diagonal)
  [k+m] decays as (1/2)^m (sub-diagonal, carry self-healing)
det(∇_a C) = ∏(b_i ⊕ c_{i-1}), P(det ≠ 0) = 2^{-n}
```

Carry-Laplacian: Δ_C f(a) = Σ_k f(a⊕e_k) · ∂C/∂a_k.
|Harm_C| ≈ 2^{n/2} — совпадает с birthday bound.

### 6.5 Carry Geometry

- Carry ball: B_r(a) = {b : d_carry(a,b) ≤ r}

```

Γ(a,b) = (a+b) ⊕ (a⊕b) — carry-коррекция
```

Γ удовлетворяет коцикловому условию: Γ(a,b) ⊕ Γ(a⊕b,c) = Γ(a,b⊕c) ⊕ Γ(b,c).

**Кривизна SHA-256:** κ = 0 на dead rounds, скачок на barrier, линейный рост далее.
λ_κ = C/4.5 ≈ 7.1 бит/раунд.

```

```
