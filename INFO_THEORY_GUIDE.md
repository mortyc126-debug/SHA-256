## 1. Что такое теория информации

Теория информации — математическая дисциплина, изучающая:
- **количественное измерение** информации,
- **пределы сжатия** данных без потерь и с потерями,
- **пределы надёжной передачи** данных через шумные каналы,
- **связь** между случайностью, неопределённостью и энергией.

Основана Клодом Шенноном в статье *«A Mathematical Theory of Communication»* (1948). Сегодня теория информации — это не только про связь: она лежит в фундаменте криптографии, машинного обучения, биоинформатики, квантовых технологий и нейронаук.

**Главная идея:** информация = снятая неопределённость. Чем менее ожидаемо событие, тем больше информации несёт сообщение о нём.

## 2. Базовые понятия

| Термин | Определение (кратко) |
|---|---|
| Источник | Случайный процесс, порождающий символы из алфавита 𝒳 |
| Канал | Механизм передачи, вносящий шум: P(y\|x) |
| Код | Отображение сообщений в последовательности символов |
| Бит | Единица информации; 1 бит = различение двух равновероятных исходов |
| Алфавит | Множество возможных символов |
| Случайная величина | Формальная модель «неопределённого исхода» |

## 3. Энтропия Шеннона

**Определение.** Для дискретной случайной величины X с распределением p(x):

```
H(X) = − Σ p(x) · log₂ p(x)
```

**Свойства:**
- H(X) ≥ 0
- H(X) максимальна при равномерном распределении: H_max = log₂|𝒳|
- H(X) = 0, если X детерминирована

**Пример.** Честная монета: H = 1 бит. Монета с p(орёл)=0.9: H ≈ 0.47 бита.

**Совместная и условная энтропии:**
```
H(X,Y) = − Σ p(x,y) log p(x,y)
H(Y|X) = H(X,Y) − H(X)
```

**Интерпретации:**
- Минимальное среднее число бит для кодирования исхода X (теорема Шеннона о кодировании источника).
- Мера «хаотичности» распределения.
- Термодинамическая аналогия: H ≡ S/k_B · ln 2.

## 4. Взаимная информация и KL-дивергенция

**Взаимная информация** I(X;Y) — сколько информации одна величина даёт о другой:

```
I(X;Y) = H(X) + H(Y) − H(X,Y) = H(X) − H(X|Y)
```

- I(X;Y) = 0 ⇔ X и Y независимы
- I(X;Y) симметрична
- Основной инструмент в представлении обучения (representation learning), InfoGAN, InfoNCE, MINE.

**KL-дивергенция** (относительная энтропия):

```
D_KL(p‖q) = Σ p(x) log (p(x)/q(x))
```

- Не симметрична, не метрика, но ≥ 0 (неравенство Гиббса).
- Центральна для вариационного вывода, VAE, обучения с подкреплением (TRPO, PPO).

**Cross-entropy:** H(p, q) = H(p) + D_KL(p‖q) — функция потерь в классификации.

## 5. Кодирование источника (сжатие)

**Теорема Шеннона о кодировании источника.** Любой источник с энтропией H может быть сжат в среднем до H бит на символ, но не ниже.

**Практические коды:**
- **Хаффман** — оптимальный префиксный код (1952).
- **Арифметическое кодирование** — практически достигает энтропийного предела.
- **LZ77/LZ78, LZW** — словарные методы; основа gzip, zstd, PNG.
- **ANS (Asymmetric Numeral Systems)** — современный стандарт (Zstandard, Facebook Zstd, JPEG-XL).

**Сжатие с потерями:**
- Скорость-искажение R(D) задаёт нижнюю границу для заданного уровня искажения D.
- Современные нейросетевые кодеки (Ballé et al., 2018 → 2025) опираются на обучаемые априорные распределения.

## 6. Канальное кодирование и пропускная способность

**Пропускная способность канала:**
```
C = max_{p(x)} I(X;Y)
```

**Теорема Шеннона о канальном кодировании.** При скорости R < C ошибку декодирования можно сделать сколь угодно малой; при R > C — нет.

**Практические коды, приближающиеся к C:**
- Турбо-коды (1993).
- LDPC (Low-Density Parity-Check) — Wi-Fi 6/7, DVB-S2, 5G data.
- Полярные коды (Arıkan, 2008) — 5G control channels.
- **Нейросетевое декодирование** (DeepPolar, 2024–2025) — ML-ассистированные декодеры.

## 7. Связь с криптографией

Криптография фундаментально опирается на теорию информации:

- **Идеальная секретность** (Шеннон, 1949): для неё требуется ключ с энтропией ≥ энтропия сообщения → one-time pad.
- **Энтропия паролей/ключей** определяет сопротивление перебору.
- **Хеш-функции** (SHA-256) — функции, эмпирически обладающие свойством «максимальной энтропии вывода» при любом входе.
- **Rényi-энтропия** H_α используется для оценки min-entropy (важно для извлечения случайности).
- **Leftover Hash Lemma** — база для рандомизаторов и KDF.

**SHA-256 в контексте ТИ:** при равномерном распределении входов выход должен быть статистически неотличим от равномерного на {0,1}²⁵⁶; это эквивалентно условию H(hash(X)) ≈ 256 бит.

## 8. Актуальные направления исследований 2025–2026

Ниже — срез активных фронтов, подтверждённых публикациями на arXiv, в Entropy (MDPI), материалами ISIT 2026 и программой JSAIT.

### 8.1. Квантовая теория информации
- **Quantum Shannon Information Theory** (arXiv:2510.07726, окт. 2025) — обобщает шенноновскую модель на квантовые каналы; предлагает «совершенный» шифр, обходящий классическую невозможность Шеннона без потери пропускной способности.
- W-состояние запутанности впервые измерено экспериментально (Kyoto/Hiroshima, сентябрь 2025) — открывает новые протоколы квантовой телепортации.
- QNodeOS — операционная система для гетерогенных квантовых сетей.
- **QKD (Quantum Key Distribution)** — интеграция с пост-квантовой криптографией.

### 8.2. Semantic Communications / 6G
- Переход от «битовой верности» к «смысловой верности».
- Кодирование задач, а не символов: отправляется только релевантная для цели получателя часть информации.
- Theoretical gap: классическое R(D) не описывает искажение *смысла*; нужны новые меры (task-relevant rate-distortion, Goal-Oriented Information Bottleneck).
- Риски: **семантические атаки** — искажение смысла без изменения битов.

### 8.3. Information Theory + Machine Learning
- **Information Bottleneck (Tishby)** остаётся активным: новые работы 2025 года связывают его с generalization bounds в transformer-моделях.
- **Shannon invariants** (arXiv:2504.15779, апр. 2025) — масштабируемая декомпозиция высокоразрядной информации (синергия/избыточность) для больших систем.
- **Open-world ML as information optimization** (arXiv:2510.15422, окт. 2025) — расширение ТИ на непрерывное обучение в нестационарной среде.
- **Neural compression** (nonlinear transform coding) — state-of-the-art в сжатии изображений/видео.

### 8.4. DNA-storage и биоинформатика
- ISIT 2026 включает DNA storage как отдельный track.
- Специфика: квартеричный алфавит {A,C,G,T}, асимметричные ошибки (вставки/удаления), композиционные ограничения.
- Разрабатываются новые коды: constrained codes для GC-balance, индельные коды (insertion/deletion codes).

### 8.5. Electromagnetic Information Theory (EIT)
- Объединяет теорию информации с уравнениями Максвелла: пропускная способность учитывает физические ограничения поля (а не только канал).
- Применение: MIMO-массивы сверхбольшой размерности, holographic surfaces, near-field communications.

### 8.6. Data Compression: Classical meets Modern
- Спецвыпуск JSAIT (приём до 31 марта 2026) — классические теории + нейросетевое сжатие + LLM-based compression (GPT как предсказатель → арифметический кодер).

### 8.7. Integrated Information Theory (IIT) и сознание
- Adversarial testing IIT vs GNWT (Nature, 2025) — эмпирические проверки теорий сознания, основанных на ТИ.
- Остаётся спорным разделом; критика в Erkenntnis (Springer, 2025).

## 9. Открытые задачи и точки роста

Перспективные направления для самостоятельных исследований:

1. **Semantic rate-distortion**: формализовать функцию искажения смысла для конкретной задачи (например, классификации или диалога LLM).
2. **ТИ анализ LLM**: измерить взаимную информацию между промптом и выходом по слоям трансформера; связать с emergent capabilities.
3. **Энтропийные оценки криптографических хешей** (в т.ч. SHA-256): статистические тесты Rényi-энтропии выхода на структурированных входах.
4. **Пост-квантовая ТИ**: пропускная способность каналов под пост-квантовыми предположениями.
5. **Compression-based ML**: использовать чистое сжатие (gzip, LZMA) как универсальный классификатор (работы 2023+ показали конкурентоспособность с BERT на малых данных).
6. **Информационно-теоретический анализ федеративного обучения**: утечка информации через градиенты (MI между градиентом и приватными данными).
7. **Каналы с обратной связью и активным выбором** (active hypothesis testing).
